# Screenshots and Visual Assets

This directory contains screenshots and visual assets for the Product Analytics Handbook.

## Screenshots

Screenshots demonstrate the application's features and user interface.

### How to Add Screenshots

1. Run the application:
   ```bash
   streamlit run streamlit_app/Product_Analytics/main.py
   ```

2. Navigate to the page you want to capture

3. Take a screenshot:
   - **Windows**: Windows Key + Shift + S
   - **macOS**: Cmd + Shift + 4
   - **Linux**: Use Screenshot tool or `gnome-screenshot`

4. Save with descriptive name:
   - `homepage.png`
   - `probability-fundamentals.png`
   - `data-visualization.png`
   - `churn-analysis.png`
   - etc.

5. Add to this directory

6. Reference in README.md:
   ```markdown
   ![Description](docs/images/filename.png)
   ```

## Image Guidelines

- **Format**: PNG or JPEG
- **Resolution**: 1920x1080 or similar (high quality)
- **File size**: Compress to < 500KB if possible
- **Content**: Clear, well-lit, readable text
- **Privacy**: No sensitive or personal data

## Logo/Banner

If creating a logo or banner:

- Keep it simple and professional
- Use consistent colors
- Make it recognizable at small sizes
- Include project name
- Consider using data visualization elements

## Tools for Creating Graphics

- **Canva** - For banners and logos
- **Figma** - For design mockups
- **GIMP** - Free image editing
- **Inkscape** - Vector graphics
- **PlotDigitizer** - For chart graphics
